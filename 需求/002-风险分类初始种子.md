# 风险分类初始种子库（珠宝行业背景）

## 元数据
- 版本: v1.0-seed
- 创建: 2026-05-23
- 机制: 动态增长，每次比对发现新类型自动追加，命中次数持续累加

## 初始分类

### 1. 交付时效
- 关键变化锚点: 交货期限、验收周期、逾期定义、宽限期
- 珠宝场景关注: 定制珠宝生产周期长、节假日大促备货时间窗口
- 命中次数: 0

### 2. 质量与鉴定标准
- 关键变化锚点: 成色、净度、克拉、切工、贵金属纯度、鉴定证书类型
- 珠宝场景关注: GIA/NGTC 等第三方认证要求变化、收货质检标准升降
- 命中次数: 0

### 3. 付款与结算
- 关键变化锚点: 付款节点、账期、保证金、佣金/扣点、手续费
- 珠宝场景关注: 高客单价对应的资金占用、分期付款条款、平台扣点变化
- 命中次数: 0

### 4. 违约责任
- 关键变化锚点: 违约金比例、赔偿上限、免责范围、不可抗力定义
- 珠宝场景关注: 高价值商品损失赔偿是否设上限、包装/运输损坏责任方
- 命中次数: 0

### 5. 知识产权与品牌
- 关键变化锚点: 商标授权范围、设计版权、图片使用权、品牌露出要求
- 珠宝场景关注: 设计师款/联名款版权归属、产品图片二次使用权限
- 命中次数: 0

### 6. 退换货与售后
- 关键变化锚点: 退货条件、换货期限、维修责任、运费承担方
- 珠宝场景关注: 定制款是否支持退换、贵金属饰品维修/翻新责任
- 命中次数: 0

### 7. 保险与风险承担
- 关键变化锚点: 运输保险、仓储保险、在途风险承担方
- 珠宝场景关注: 高价值货品物流必须投保、投保金额是否足额
- 命中次数: 0

### 8. 运输与物流
- 关键变化锚点: 承运方资质、配送时效、签收规则、保价要求
- 珠宝场景关注: 须用特种物流（武装押运/专车）、保价金额上限
- 命中次数: 0

### 9. 价格调整机制
- 关键变化锚点: 调价通知期限、原材料价格联动条款、平台活动强制降价
- 珠宝场景关注: 金价/钻石价格波动大，原料联动定价条款是否公平
- 命中次数: 0

### 10. 数据与隐私
- 关键变化锚点: 客户数据使用范围、数据共享限制、数据安全责任
- 珠宝场景关注: 高净值客户隐私保护、消费数据不得用于竞品分析
- 命中次数: 0

## 扩展机制
- JSON 存储路径: `data/risk_taxonomy.json`
- 新增类型自动分配 ID，从 11 开始递增
- 每次 diff 运行时，`changes[].risk_categories` 引用的分类命中次数 +1
- 命中次数 > 10 的分类标注 `attention_level: high`，触发二次校验建议
